# Contributing to Maple

Thank you for your interest in contributing to Maple! We welcome contributions from everyone.

## how to contribute

1.  **Fork the repository** and create your branch from `main`.
2.  **Clone the repository** to your local machine.
3.  **Create a new branch** for your feature or bug fix: `git checkout -b my-new-feature`
4.  **Make your changes** and test them.
5.  **Commit your changes** with clear commit messages.
6.  **Push to your fork** and submit a **Pull Request**.

## Reporting Bugs

If you find a bug, please open an issue on GitHub. include as much detail as possible:

*   Steps to reproduce the bug
*   Expected behavior
*   Actual behavior
*   Screenshots (if applicable)
*   Your operating system and environment

## Development

Please refer to the `README.md` for setup instructions.

## Code Style

*   Keep code clean and readable.
*   Follow existing coding style and conventions.
*   Write meaningful variable and function names.
